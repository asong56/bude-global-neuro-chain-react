# Contributing to Bude Global Neuro-Chain

Thanks for your interest in contributing. This project welcomes developers, designers, historians, educators, researchers, writers, and curious thinkers alike.

## How to Contribute

1. **Fork** the repository
2. **Create a branch**: `git checkout -b feature/your-idea`
3. **Make your changes** — keep them focused and incremental
4. **Test locally**: `npm run dev`
5. **Submit a Pull Request** — describe what you changed and why

## Ways to Help

- **Code** — performance, refactoring, new interactions, bug fixes
- **UI/UX** — layout, zooming, filtering, visual metaphors
- **Content** — add innovation nodes, improve relationships, fix categorization
- **Docs** — README clarity, comments, accessibility, translations
- **Community** — share the project, help new contributors get started

## First-Time Contributors

Start small:

- Fix a typo
- Add a single innovation node
- Ask a question in an issue

Look for issues labeled `good first issue` or `help wanted`.

## Philosophy

- Be respectful and constructive
- Assume good intent
- Ask questions early
- Share ideas openly

A maintainer will review your PR and may suggest improvements — that's collaboration, not criticism.
